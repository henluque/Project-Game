Este projeto tem o intuito de organizar e versionar o nosso projeto da matéria de Multimídia, cujo objetivo é criar um jogo usando a linguagem LUA.

---

##🎮 Gameplay

| **Ação** | **Teclado**  | **Controle**      |
| :------- | :----------: | ----------------: |
| Mover    | A D / ← →    | Analógico / D-pad |
| Pular    | W / ↑        | Botão A           |
| Ataque   | Z / K        | Botão X           |
